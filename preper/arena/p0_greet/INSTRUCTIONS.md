# p0 - Warm-up: greet (implement)

A 5-minute warm-up to confirm the loop works. Implement `greet` in `solution.py`.

## Contract
`greet(name: str) -> str`
- If `name` is a non-empty string, return `"Hello, {name}!"`.
- If `name` is empty (`""`), return `"Hello there!"`.

## The loop
1. Start a timer:      `python arena/timer.py 5`   (in another terminal)
2. Edit               `arena/p0_greet/solution.py`
3. Debug locally:      `python arena/run.py p0`
4. Score it:           `python arena/run.py p0 --scored`
